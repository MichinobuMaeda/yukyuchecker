## 利用規約

有休チェッカーは以下の条件により無料で利用できます。

1. セキュリティの強度とパフォーマンスは利用するクラウド基盤に依存します。
2. パスワードを使ってログインする場合、大文字、小文字、数字、記号のすべてを含む10文字以上のパスワードとしてください。
3. 利用するクラウド基盤の仕様により、管理者は利用者のパスワードを知ることができません。パスワードを失念された場合は「利用開始」の案内に従ってご自身でパスワードを再設定してください。
4. 個人で運営しているため、運営者の都合により予告なしでアプリの運用を終了する場合があります。

### プライバシーポリシー

有休チェッカー（以下「本サービス」）は、利用者の個人情報の保護に配慮し、以下の方針で取り扱います。

1. 取得する情報
    - 利用者の認証に必要な情報（メールアドレス等）
    - 利用者が本サービスに入力した勤務・休暇等のデータ
    - 本サービスの運用に必要なログ情報
2. 利用目的
    - 利用者認証および本人確認のため
    - 有休管理機能を提供するため
    - 本サービスの保守、障害対応、品質改善のため
3. 情報の保管と管理
    - 利用者の認証には Firebase Auth を利用し、利用者データは Cloud Firestore に保管します。
    - 管理者は、運用上必要な範囲で、利用者のパスワードを除くデータを参照できる場合があります。
    - 管理者以外の利用者は、本人が入力したデータのみ参照できます。
4. 第三者提供
    - 法令に基づく場合を除き、利用者の個人情報を第三者へ提供しません。
    - 本サービスの提供に必要な範囲で、Google Cloud の Firebase を利用します。
5. 利用者による削除
    - 利用者が自身のアカウントを削除した場合、当該利用者に紐づくデータは削除されます。
    - 削除されたデータは復元できません。
6. セキュリティ
    - 本サービスは、利用するクラウド基盤が提供するセキュリティ機能を前提として運用します。
7. プライバシーポリシーの変更
    - 本ポリシーは、必要に応じて予告なく改定する場合があります。
8. お問い合わせ
    - 本ポリシーに関するお問い合わせ先は、<https://pages.michinobu.jp/> をご参照ください。
