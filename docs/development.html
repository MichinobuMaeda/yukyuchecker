Development

Prerequisites

  • Git
  • Flutter >= 3.41.5
  • Node.js >= 24
  • Java >= JDK 11

Get started

git clone git@github.com:MichinobuMaeda/yukyuchecker.git
cd yukyuchecker
flutter pug get
npm i
npm i --prefix functions
num run dev
  • http://localhost:8000/ -- Web Server in debug mode
  • http://localhost:4000/ -- Firebase Emulator Suite UI
